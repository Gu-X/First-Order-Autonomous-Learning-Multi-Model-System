# First-Order-Autonomous-Learning-Multi-Model-System
This is the python code of the first-order autonomous learning multi-model system (python 3.6). 
This work is described in:
==================================================================================================
P. P. Angelov, X. Gu, J. C. Principe, "Autonomous learning multi-model systems from data streams," 
IEEE Transactions on Fuzzy Systems, vol. 26(4), pp. 2213-2224, 2018.
